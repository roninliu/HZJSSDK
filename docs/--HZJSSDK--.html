  1. 生成随机数

文件类名：Utils.js
示例：

  1. var random = new Utils();       
  2. /*生成0到1之间的随机数*/
  3. var r1 = random.getIntervalRandom();
  4. console.log("0到1之间的随机数:",r1);
  5. /*生成1到100的随机数*/
  6. var r2 = random.getIntervalRandom(1,100);
  7. console.log("1到100的随机数",r2);

输出：

  1. 0到1之间的随机数: 0.5765791456215084
  2. 1到100的随机数 36
